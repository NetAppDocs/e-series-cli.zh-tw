= 停止Volume群組定位
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
「最大磁碟區群組定位」命令會關閉「最小磁碟區群組定位」命令所開啟磁碟機上的指示燈。



== 支援的陣列

此命令適用於任何個別儲存陣列、包括E2700、E5600、E2800、E5700、 EF600和EF300陣列、只要安裝所有SMcli套件即可。



== 角色

若要在E2800、E5700、EF600或EF300儲存陣列上執行此命令、您必須具有儲存管理員角色。



== 語法

[listing]
----
stop volumeGroup locate
----


== 參數

無。



== 最低韌體層級

6.16
