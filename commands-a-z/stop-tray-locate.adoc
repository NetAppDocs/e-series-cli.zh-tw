= 停止紙匣定位
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用「最小化紙匣定位」命令、會關閉以「開始紙匣定位」命令開啟之紙匣上的指示燈。



== 支援的陣列

此命令適用於任何個別儲存陣列，包括 E4000 ， E2700 ， E5600 ， E2800 ， E5700 ， EF600 和 EF300 陣列，只要安裝了所有 SMCLI 套件即可。



== 角色

若要在 E4000 ， E2800 ， E5700 ， EF600 或 EF300 儲存陣列上執行此命令，您必須具有儲存管理員角色。



== 語法

[source, cli]
----
stop tray locate
----


== 參數

無。



== 最低韌體層級

6.10.
