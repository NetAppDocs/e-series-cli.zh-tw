= 啟動非同步鏡射
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
「activate storageArray facture'命令會啟動「非同步鏡射」功能。



== 支援的陣列

此命令適用於任何個別儲存陣列、包括E2700、E5600、E2800及E5700陣列、 只要安裝所有SMcli套件、



== 角色

若要在E2800或E5700儲存陣列上執行此命令、您必須擁有儲存管理員角色。



== 背景

啟動非同步鏡射功能之後、您必須設定非同步鏡射群組和非同步鏡射配對。



== 語法

[listing]
----
activate storageArray feature=asyncRemoteMirror
----
[listing]
----
activate storageArray feature=asyncMirror
----


== 參數

無。
